# hackathon

このリポジトリには、2022年7月16日に開催された福井オープンデータハッカソンのオープンデータ、プレゼン動画、およびソースコードが含まれています。

## ライブデモ

- **[プレゼン動画](https://code4fukui.github.io/hackathon/movie.html)**: 各チームの最終プレゼン動画を視聴できます。このページはプロジェクトデータを動的に読み込み、動画とチームのスライドやGitHubリポジトリへのリンクを併せて表示します。プレゼン内容には「キャンプパス」モバイルアプリや鉄道トンネルを活用した観光プロジェクトなどがあります。
- **[CSVデータビューア](https://code4fukui.github.io/hackathon/)**: ハッカソンの生のプロジェクトデータをインタラクティブなテーブル形式で表示するビューアです。`csv-viewer.js`ウェブコンポーネントを使用しています。

## オープンデータ

イベントのデータはCSV形式で提供されており、各チームのリソースへのリンクが含まれています。

- **[2022-07-16 福井観光アイデアソン＆ハッカソンデータ](https://code4fukui.github.io/hackathon/20220716.csv)**: チーム名、プロジェクトタイトル、スライド・GitHubリポジトリ・プレゼン動画のURLが含まれています。

## 技術概要

このリポジトリは、ハッカソンの結果を整理・表示するためにシンプルなウェブ技術とユーティリティスクリプトを使用しています。

- **`index.html`**: 生のCSVデータをHTMLテーブルとしてレンダリングするビューア。
- **`movie.html`**: `20220716.csv`を解析し、プロジェクト動画、タイトル、リソースリンクのギャラリーを動的に生成するページ。
- **`20220716/makethumbnail.sh`**: FFmpegを使用してプレゼン動画（`.mov`）の最初のフレームを抽出し、サムネイル（`.jpg`）として保存するユーティリティスクリプト。
- **`makemd.js`**: CSVを読み込み、プロジェクトのリストとGitHubリンクをMarkdown形式でコンソールに出力するシンプルなスクリプト。
